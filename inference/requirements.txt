# DARWIN inference layer dependencies.
#
# IMPORTANT: install with python3.11 (/opt/homebrew/bin/python3.11).
# Python 3.14 lacks prebuilt wheels for mlx and its native deps; 3.11 is the
# supported interpreter for everything MLX-related on this machine.
#
#   /opt/homebrew/bin/python3.11 -m venv .venv
#   .venv/bin/pip install -r inference/requirements.txt
#
# MLX runs on the Apple GPU via Metal (Apple Silicon only).
#
# SUPPLY-CHAIN: direct deps are pinned to EXACT versions (`==`), not floating
# `>=` lower bounds. A `>=` range silently pulls whatever the newest release is at
# install time — so a single compromised or yanked PyPI upload lands in the
# inference process (which loads models and runs on-device) with no review. These
# pins are the known-good versions from the verified python3.11 venv. To upgrade
# deliberately: bump a pin, reinstall, test, commit. To refresh from a working
# venv: `.venv/bin/pip freeze | grep -iE '<the names below>'`.

# server.py works (and py_compile passes) WITHOUT this installed — absent =>
# op=describe_image honestly reports "vision-language model not available" and
# the daemon falls back to OCR/classification. NOT a hard dep: do NOT let its
# absence break the build. It pulls a multi-GB Qwen2-VL-class model download
# gated on enough RAM, which is why the vision-describe feature ships OFF/opt-in.
# Install ONLY when enabling that feature:
#   .venv/bin/pip install 'mlx-vlm>=0.1'
# HONESTY: the VLM runs ON-DEVICE via MLX — the image pixels never leave the
# device. Description quality is device/runtime-gated and is never claimed
# measured here.
# mlx-vlm>=0.1   # (intentionally commented out: optional, import-guarded extra)

# OPTIONAL on-device TEXT->IMAGE GENERATOR (op=generate_image). IMPORT-GUARDED:

# surfaces an honest "the on-device image model isn't set up" message. NOT a

#   .venv/bin/pip install 'mflux>=0.4'
# HONESTY: generation runs ON-DEVICE via MLX — the prompt AND the generated
# pixels never leave the device (there is NO cloud image API). Image quality and
